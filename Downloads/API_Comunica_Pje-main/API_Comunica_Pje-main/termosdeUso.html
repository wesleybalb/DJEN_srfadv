2. DEFINIÇÕES

3. NATUREZA E FINALIDADE DA FERRAMENTA

3.1. Função da Ferramenta

A Ferramenta é um intermediário tecnológico que facilita o acesso aos dados públicos disponibilizados pelo CNJ através de sua API oficial. A Ferramenta NÃO CRIA, MODIFICA OU VALIDA qualquer informação judicial.

3.2. Fonte dos Dados

Todos os dados apresentados são exclusivamente originários da API oficial do CNJ. A Ferramenta apenas:

  • Consulta a API CNJ
  • Organiza os dados retornados
  • Exporta em formato Excel

3.3. Limitações Técnicas

A Ferramenta está sujeita às limitações, disponibilidade e funcionamento da infraestrutura tecnológica do CNJ.
